Лекція 2.1: Алгоритми і структури даних: списки

Вивчення основ списків у програмуванні: створення, маніпуляція, використання.

Списки є основною структурою даних, яка використовується для зберігання та організації колекцій елементів. Вони можуть бути реалізовані різними способами, такими як масиви або зв'язані списки. Списки дозволяють ефективно додавати, видаляти та шукати елементи.

Вступ

У програмуванні списки є однією з основних структур даних. Вони дозволяють зберігати множину елементів в певному порядку, що дозволяє ефективно маніпулювати даними. Списки можна використовувати для зберігання будь-якого типу даних і виконання різних операцій над ними, таких як додавання, видалення, пошук та сортування елементів.

У цій лекції ми розглянемо основи списків у програмуванні, їх створення, маніпулювання даними та використання списків в реальних задачах.

1. Що таке список?

Список — це структура даних, яка складається з впорядкованого набору елементів. Елементи списку можуть бути будь-якими типами даних, такими як числа, рядки або навіть інші списки. Всі елементи списку зберігаються в певному порядку, що дозволяє зручно маніпулювати ними.

Списки можуть бути реалізовані у двох основних варіантах:

  • Масив: статичний список, де розмір фіксований, і доступ до елементів здійснюється за індексом.
  • Зв'язаний список: динамічна структура, де елементи (вузли) зберігаються в пам'яті не підряд, а кожен елемент має посилання на наступний.

2. Види списків

Однозв'язаний список:

  • Кожен елемент (вузол) містить значення та посилання на наступний елемент.
  • Можна йти через список тільки в одному напрямку (від початку до кінця).

Двозв'язаний список:

  • Кожен елемент має два посилання: на наступний та попередній елемент.
  • Дозволяє проходити по списку в обох напрямках (від початку до кінця і назад).

Масив списку:

  • Це просто масив елементів, де всі дані зберігаються у безперервному блоці пам'яті.
  • Має обмеження на розмір, який визначається при створенні масиву.

3. Створення списку

У різних мовах програмування списки можуть бути реалізовані по-різному. Розглянемо створення списку в Java.

int[] numbers = new int[5];  // масив, що містить 5 елементів
numbers[0] = 10;
numbers[1] = 20;
numbers[2] = 30;
numbers[3] = 40;
numbers[4] = 50;

Створення списку з використанням класу ArrayList:

import java.util.ArrayList;

ArrayList numbers = new ArrayList<>();
numbers.add(10);  // додавання елемента
numbers.add(20);
numbers.add(30);
numbers.add(40);
numbers.add(50);

Використання зв'язного списку:

class Node {
    int data;
    Node next;

    Node(int data) {
        this.data = data;
        this.next = null;
    }
}

public class LinkedList {
    Node head;

    void addNode(int data) {
        Node newNode = new Node(data);
        if (head == null) {
            head = newNode;
        } else {
            Node current = head;
            while (current.next != null) {
                current = current.next;
            }
            current.next = newNode;
        }
    }
}

4. Маніпуляція списками

Однією з основних переваг списків є можливість ефективно маніпулювати елементами. Ось деякі з операцій, які можна виконувати над списками:

Додавання елементів:

У масиві елементи додаються в кінці або за індексом.

У списку можна додавати елементи в будь-яку точку, не потребуючи значних витрат пам'яті.

Видалення елементів:

Видалення елементів з масиву зазвичай вимагає пересування всіх елементів після видаленого.

У зв'язному списку видалення елементів здійснюється шляхом перепризначення посилань між елементами.

Пошук елементів:

У масиві можна здійснювати пошук елементів за індексом.

У списках пошук зазвичай здійснюється шляхом послідовного перебору елементів, що може бути менш ефективним, ніж пошук за індексом у масиві.

Приклад маніпуляції списком у Java:

ArrayList numbers = new ArrayList<>();
numbers.add(10);
numbers.add(20);
numbers.add(30);

// Видалення елемента за індексом
numbers.remove(1);  // видалить елемент 20

// Додавання елемента в певну позицію
numbers.add(1, 25);  // вставить 25 на 1-й позиції

// Отримання елемента за індексом
int element = numbers.get(1);  // отримаємо елемент на 1-й позиції (25)

5. Використання списків

Списки використовуються у багатьох типах задач. Вони ефективні при роботі з динамічними даними, коли розмір списку може змінюватися під час виконання програми. Ось кілька прикладів використання списків:

Перебір елементів:

Перебір елементів списку — це найпоширеніша операція при роботі з даними. Для цього можна використовувати цикл for або for-each.

for (int number : numbers) {
    System.out.println(number);
}

Зберігання історії:

Списки часто використовуються для зберігання історії дій користувача, таких як зміни в документах або вибір користувача в інтерфейсі програми.

Алгоритми сортування та пошуку:

Списки можна використовувати в алгоритмах сортування (наприклад, сортування бульбашкою, сортування вставками) та пошуку (наприклад, лінійний пошук, бінарний пошук).

Стек та черга:

Списки можуть бути використані для реалізації структури даних «стек» (LIFO — останнім прийшов, першим пішов) або «черга» (FIFO — першим прийшов, першим пішов).
