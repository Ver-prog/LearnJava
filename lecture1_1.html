Лекція 1.1: Основи програмування: змінні

Огляд базових понять змінних у програмуванні: визначення, типи, оголошення.

Змінні — це іменовані області пам'яті, в яких зберігаються дані, що можуть змінюватися під час виконання програми. Змінні мають типи, які визначають тип даних, що можуть бути в них збережені (наприклад, ціле число, рядок тексту, логічне значення).

Для оголошення змінної в різних мовах програмування використовують різні синтаксиси. Наприклад, у C++ можна оголосити змінну так: int a;, де int — тип змінної, а a — її ім'я.

Що таке змінні?

Змінні — це основні контейнери для зберігання інформації в будь-якій програмі. Уявіть собі, що змінна — це коробка, у якій можна тримати значення: число, текст, істину чи хибу, або навіть щось складніше. Важливість змінних важко переоцінити: вони дозволяють програмам "пам’ятати" дані та працювати з ними.

Наприклад:

name = "Олена"
age = 25
is_programmer = True

У цьому прикладі ми створили три змінні: name, яка містить текст, age, що зберігає число, та is_programmer, яка вказує на істину чи хибу (логічний тип).

Змінні — це основа програмування. Завдяки ним ми можемо створювати складні алгоритми, зберігаючи й обробляючи величезну кількість інформації.

Навіщо потрібні змінні?

Уявіть, що ви хочете написати програму для калькулятора. Вам потрібно, щоб користувач ввів два числа, і програма виконала з ними операцію. Змінні дозволяють зберегти ці числа та використати їх у потрібний момент.

x = 10
y = 5
sum = x + y
print(sum)

У цьому прикладі змінні x і y зберігають числа, які ми додаємо, а результат зберігається в змінній sum.

Крім того, змінні дають змогу:

  • уникати дублювання коду;
  • робити програми гнучкими;
  • створювати чіткі й зрозумілі алгоритми.

Типи змінних

Різні мови програмування пропонують різні типи змінних. Тип змінної визначає, який саме вид даних вона може зберігати. Основні типи змінних:

Числові типи:

  • Цілі числа (“integer” або “int”) — для збереження цілих чисел, наприклад, 5, -10.
  • Дробові числа (“float” або “double”) — для чисел з плаваючою точкою, таких як 3.14, -0.001.

Текстові типи:

  • Рядки (“string”) — для збереження тексту. Наприклад: "Привіт, світ!".

Логічні типи:

  • Булеві значення (“boolean” або “bool”) — можуть бути лише True або False.

Складені типи:

  • Масиви або списки (“lists”, “arrays”) — для зберігання множин даних, наприклад, [1, 2, 3].
  • Словники (“dictionaries”) — для пар “ключ-значення”, наприклад, {"ім'я": "Олена", "вік": 25}.

Різні мови можуть мати й інші унікальні типи. Наприклад, у Python є тип set для множин, а в JavaScript — undefined та null.

Як оголошувати змінні?

Оголошення змінних — це процес створення змінної та (зазвичай) призначення їй значення. Правила оголошення змінних залежать від мови програмування, але існують спільні принципи:

Ім’я змінної:

  • Має бути зрозумілим і описовим, наприклад, user_age, а не x.
  • Не повинно починатися з цифри (наприклад, 1name — недопустимо).
  • Не має містити пробілів або спеціальних символів, окрім підкреслення _.

Присвоєння значення:

Змінній присвоюється значення за допомогою оператора =.

У деяких мовах потрібно вказувати тип змінної під час її оголошення. Наприклад, у Java:

int age = 25;
String name = "Олена";

У мовах із динамічною типізацією, таких як Python, цього робити не потрібно:

age = 25
name = "Олена"

Практичні поради для роботи зі змінними

  • Вибирайте зрозумілі імена: Гарне ім’я змінної робить код читабельним і зрозумілим. Наприклад, замість a, використовуйте user_name.
  • Дотримуйтеся стилю написання:
    • CamelCase: userName (часто використовується в JavaScript).
    • snake_case: user_name (популярний у Python).
  • Уникайте магічних чисел: Замість того, щоб писати числа без пояснень у коді, використовуйте змінні:
interest_rate = 0.05
balance = 1000
final_amount = balance * (1 + interest_rate)

Коментуйте код: Якщо змінна має складне призначення, поясніть це в коментарях.

Висновок

Змінні — це основа, без якої неможливо уявити жодну програму. Вони є ключовим інструментом для створення ефективного, гнучкого й зрозумілого коду. Знання основ змінних допомагає закласти міцний фундамент для подальшого вивчення програмування. Тому не бійтеся експериментувати, обирайте зручні імена та вивчайте особливості кожної мови програмування. Адже змінні — це той елемент, який дає програмам "життя".
