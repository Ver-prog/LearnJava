Лекція 1.2: Основи програмування: оператори

Введення в оператори у програмуванні: арифметичні, логічні, порівняння.

Оператори - це символи або ключові слова, які використовуються для виконання операцій над змінними та значеннями. Вони можуть змінювати значення змінних, порівнювати їх, виконувати математичні обчислення та багато іншого. Оператори є основою для виконання логіки програми.

1. Що таке оператори?

Оператори — це символи або ключові слова, які виконують певні операції над змінними або значеннями. Вони можуть змінювати значення змінних, порівнювати їх, виконувати математичні обчислення та багато іншого. Оператори є основою для виконання логіки програми.

2. Класифікація операторів

Оператори можна поділити на кілька основних категорій, залежно від їхнього призначення та типу операцій:

2.1. Арифметичні оператори

Ці оператори використовуються для виконання математичних операцій. Вони включають:

  • + — додавання
  • - — віднімання
  • * — множення
  • / — ділення
  • % — залишок від ділення (модуль)

Приклад:

a = 5
b = 3
c = a + b  # c = 8
d = a * b  # d = 15
e = a % b  # e = 2 (залишок від ділення 5 на 3)

2.2. Оператори порівняння

Ці оператори дозволяють порівнювати значення змінних або виразів. Вони повертають булеве значення (True або False).

  • == — рівність
  • != — нерівність
  • > — більше
  • < — менше
  • >= — більше або рівне
  • <= — менше або рівне

Приклад:

x = 10
y = 20
result = x < y  # result = True

2.3. Логічні оператори

Логічні оператори дозволяють поєднувати кілька умов або порівнянь.

  • and — логічне "і"
  • or — логічне "або"
  • not — логічне "не"

Приклад:

a = 5
b = 10
if a < 10 and b > 5:
    print("Умова виконується")
# Виведе: "Умова виконується"

2.4. Оператори присвоєння

Оператори присвоєння використовуються для того, щоб призначити значення змінним.

  • = — просте присвоєння
  • += — додавання і присвоєння
  • -= — віднімання і присвоєння
  • *= — множення і присвоєння
  • /= — ділення і присвоєння

Приклад:

x = 5
x += 3  # x = x + 3, результат x = 8
x *= 2  # x = x * 2, результат x = 16

2.5. Оператори інкременту та декременту

Ці оператори дозволяють збільшувати або зменшувати значення змінної на одиницю.

  • ++ — інкремент (збільшення на 1)
  • -- — декремент (зменшення на 1)

Приклад:

x = 5
x++  # x = 6 (збільшення на 1)
x--  # x = 5 (зменшення на 1)

Проте варто зазначити, що в Python оператори ++ та -- відсутні, і замість цього використовуються оператори += 1 та -= 1.

2.6. Оператори типу is і in

Ці оператори використовуються для порівняння і перевірки належності елементів до колекцій.

  • is — перевірка на ідентичність (чи є два об'єкти однаковими)
  • in — перевірка на належність елемента до колекції (наприклад, списку чи рядка)

Приклад:

x = [1, 2, 3]
y = 2
print(y in x)  # True, бо 2 є елементом списку x

3. Оператори в умовних конструкціях

Оператори часто використовуються в умовних конструкціях для вибору виконання певної частини коду залежно від умов.

Приклад умовного оператора if:

a = 10
b = 20
if a < b:
    print("a менше за b")
else:
    print("a не менше за b")

4. Оператори в циклах

Оператори також застосовуються в циклах для виконання операцій на кожній ітерації.

Приклад циклу for:

for i in range(5):
    print(i)  # Виведе числа від 0 до 4

5. Пріоритет операторів

Важливо розуміти, що деякі оператори мають вищий пріоритет ніж інші. Наприклад, оператори множення та ділення мають вищий пріоритет за додавання та віднімання. Для уникнення помилок та для зручності читання коду використовуються круглі дужки.

Приклад:

result = 2 + 3 * 4  # результат буде 14, а не 20
result = (2 + 3) * 4  # результат буде 20

6. Висновки

Оператори є основою програмування, оскільки саме з їхньою допомогою відбуваються всі обчислення та прийняття рішень у програмі. Розуміння їхнього використання допомагає створювати ефективні та точні програми. У процесі навчання важливо практикуватися у використанні операторів і застосовувати їх для розв'язування різних задач.

Завдання для самостійної роботи:

  • Створіть програму, яка запитує два числа у користувача, порівнює їх і виводить на екран, яке з чисел більше.
  • Напишіть програму, яка обчислює суму всіх чисел від 1 до 100 за допомогою циклу.
  • Використовуючи оператори, створіть програму, яка обчислює квадратний корінь з числа і виводить результат.
